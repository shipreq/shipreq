What is the bare minimum required?
==================================

* Requirement Storage
  * Req CRUD
* Requirement Organisation
  * Separate requirements by type (ie. MF, FR, CO) - else TMI and unusable
  * Link to MF.
  * Link to component/tag.
* Version scoping
  * Allocate req to version.
  * Allocate MF to version.
  * Infer req version from linked MF.
