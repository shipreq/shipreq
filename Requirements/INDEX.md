  * Massively incomplete. Abandoned.

Post UCE 1.0
============

`feature-organisation.ods`
  * Supplimental analysis of the organisation feature. Contains:
    * Analysis of envisioned org-units and attributes.
    * Analysis of org-units relationships.

`major_features.ods`
  * List of major features envisioned.
  * Version and analysis/impl status for each feature.
  * Analysis of feature intersection.

`prelim-nnn.odt`
  * Preliminary requirements gathering interviews.
  * Actually: elicitation, analysis.

`project-attributes.ods`
  * Top 5 project attributes.
  * Definition and analysis of each attribute.
  * Prioritisation of attributes.

`reqs.ods`
  * Intermediary location of requirements analysis.
  * Extraction of initial requirements from `prelim-nnn` files.
