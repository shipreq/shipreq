# ShipReq blog

### Usage

Run `make` to see a list of commands.

### Blogging

1. Drop a new `.mdx` file into `content/posts`
2. Make sure it has all the frontmatter fields as other posts
3. Commit and deploy

### Releasing

1. Run `make prod`
1. View it at `http://localhost:9000` and make sure you're happy with it
1. Run `make ci`
