# ShipReq blog

### Commands

* `make dev`        - Serve in dev-mode at http://localhost:8000/ and auto-reload on change
* `make dev-build`  - Builds the site minified in `./public`
* `make prod-build` - Builds the site for production in `./public`
* `make prod`       - Builds and serves production-mode at http://localhost:9000/


### Blogging

1. Drop a new `.mdx` file into `content/posts`
2. Make sure it has all the frontmatter fields as other posts
3. Commit and deploy
