What's a project template?
ShipReq is very flexible. You can configure it to behave in a way that best suits your processes and needs. Rather than having to manually specify all of that configuration by yourself, you can start a project off of a pre-configured template. Templates are just starting points. After creating a project you're free to completely change the configuration as you see fit.
